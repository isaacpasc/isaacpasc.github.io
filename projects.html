Dashify

Dashify is an ios client for web hosting service, Netlify. In this project I learned to create generic api calls with async/await and combine, authentication and preventing CSRF attacks, and WebViews with javascript. The project also uses keychain to store login tokens. The login process for Dashify uses a WebView to present a login screen for Netlify users. This WebView had some challenges to properly confrom to AppStore Review Guidelines. One issue was that Netlify offers a 'Pro' plan but Dashify does not offer an in-app-purchase method for buying this subscription. The only way around this rule is a 'free standalone app acting as a stand-alone companion to a paid web based tool'. The only issue here is that there must be not one call to action to purchase a 'Pro' Netlify account from within Dashify. This meant that I had to lock down the login experience in Dashify to only show exactly what I want and nothing more. I was able to inject javascript at the end of each document to successfully remove all upgrade buttons. And if the user ever tries to leave the login process, the WebView will clear all cookies to sign them out and send them back to login. Dashify Offical Website or find it on the AppStore.

Data Query

Data Query is a command line application using c++ with a Makefile. It querys large css files based on parameters. These queries use several data structures (custom made, no libraries) such as hashtables, BST, and graphs in adjacency list. These data structes are traversed using Floyd Warshall Algorithm and DFS (again, no libraries are used).

Compression Algorithm

Compression Algorithm is a command line application using c++ with a Makefile. It encodes or decodes any text file with ASCII characters by using either insertion sort or merge sort to build a binary search tree and traverse the tree to encode/decode.
