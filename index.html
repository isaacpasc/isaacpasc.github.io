It features many swift fundementals that are key for success in large scale applications. These fundementals include: protocals, delegate pattern, image caching, generic api calls with combine, pagination, TableView/ListView, and much more!

Data Query

Data Query is a command line application using c++ with a Makefile. It querys large css files based on parameters. These queries use several data structures (custom made, no libraries) such as hashtables, BST, and graphs in adjacency list. These data structes are traversed using Floyd Warshall Algorithm and DFS (again, no libraries are used).

Compression Algorithm

Compression Algorithm is a command line application using c++ with a Makefile. It encodes or decodes any text file with ASCII characters by using either insertion sort or merge sort to build a binary search tree and traverse the tree to encode/decode.
